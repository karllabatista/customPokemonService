2.Node js

Instale a versão LTS do Node.js
Verifique com:

node -v
npm -v

3. definir versao do node especificida
nvm (Node Version Manager) → permite instalar e trocar versões facilmente:
# instalar nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
# instalar versão LTS do Node
nvm install --lts

# instalar versao especifica
nvm install 18
# usar uma versão específica
nvm use 18

4. gerenciador de pacotes
usado: npm

init  project:
npm init -y

5. instalar dependencias principais
graphql:
npm install graphql

Apollo Server (mais usado, facilita setup de GraphQL):
